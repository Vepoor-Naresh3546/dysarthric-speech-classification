# Dysarthric Speech Classification using TORGO Dataset

## Overview

This project focuses on classifying dysarthric and control speech samples using machine learning techniques. The system extracts audio features from the TORGO dataset and predicts speech disorders through multiple classifiers.

---

## Features

- Audio preprocessing and augmentation
- Feature extraction using Librosa
- Machine Learning models:
  - Random Forest
  - Support Vector Classifier (SVC)
  - Multi-layer Perceptron (MLP)
  - Bagging Classifier
- GUI interface using Tkinter
- Performance evaluation metrics

---

## Dataset

TORGO Dataset containing four classes:

- F_Con
- F_Dys
- M_Con
- M_Dys
